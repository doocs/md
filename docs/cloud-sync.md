# 云同步说明

登录账户后，可将部分编辑器偏好同步到云端，并在其他设备上恢复。同步采用**最后写入优先（LWW）**策略：同一设置项以更新时间较新的为准。

## 会同步的内容

以下偏好会写入云端（不含密钥与敏感信息）：

| 类别              | 说明                                                                                                                 |
| ----------------- | -------------------------------------------------------------------------------------------------------------------- |
| **主题与样式**    | 当前主题、各主题独立配置（字体、字号、主题色、代码块主题、标题样式等）、首行缩进、两端对齐、引用样式、图例、预览宽度 |
| **编辑器 / UI**   | 编辑区左右位置、AI 工具栏显示、视图模式、预览设备、复制模式、排序方式、图片转存开关、滚动同步                        |
| **AI 非敏感参数** | AI 类型、温度、最大 token（**不含 API Key 与模型名称**）                                                             |
| **自定义内容**    | 自定义 CSS、模板、自定义组件、快捷命令                                                                               |

文章正文与草稿默认通过「云文档」单独管理，不在上述偏好同步范围内。

## 不会同步的内容（仅保存在本机）

出于安全考虑，以下内容**不会**上传到云端，换设备后需重新配置：

| 类别           | 说明                                                                                        |
| -------------- | ------------------------------------------------------------------------------------------- |
| **图床配置**   | GitHub、阿里云 OSS、腾讯云 COS、七牛、MinIO、S3、Telegram 等各图床的 Token、密钥、Bucket 等 |
| **AI 密钥**    | OpenAI / 其他提供商的 API Key、模型选择                                                     |
| **运行时状态** | 临时 UI 状态、未保存的本地缓存等                                                            |

## 使用建议

1. **换设备前**：确认已手动同步或等待自动同步完成（账户面板可查看同步状态）。
2. **图床与 AI**：在新设备登录同一账户后，进入「图床配置」与「AI 设置」重新填写密钥。
3. **冲突**：若两台设备同时修改同一偏好，后保存的一方会覆盖另一方，无合并对话框。
